# File Format Validator

A web application for processing and validating document files (DOCX).

## Features

- Document upload and processing
- DOCX file analysis
- Modern React frontend with TypeScript
- Express.js backend

## Tech Stack

- Frontend:
  - React
  - TypeScript
  - Vite
  - Tailwind CSS
- Backend:
  - Node.js
  - Express
  - TypeScript

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone [your-repo-url]
cd FileFormatValidator
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:3000`

## Development

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Start production server

## Deployment

This project is configured for deployment on Vercel. The `vercel.json` configuration is included in the repository.
